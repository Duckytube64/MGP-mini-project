This erosion simulation is based on the following paper:
https://www.firespark.de/resources/downloads/implementation%20of%20a%20methode%20for%20hydraulic%20erosion.pdf

CONTROLS
left mouse button drag: move model horizontally
right mouse button drag: move model vertically
scroll: zoom
space: pause/play simulation

We use normalized heightmaps of 256x256 png's.
If you can find a way to access the assets folder in the unity build, you can add your own in assets\resources\heightmaps\
Follow the name convention of 'HM_' + number.
